# girlwhodrawsandstuff.github.io

Personal portfolio website for Varsha Singh — Senior Frontend Engineer at Razorpay, based in Bangalore, India.

## What's inside

The site is a single-page React app with three sections:

- **Portfolio** — a showcase of projects and work
- **Experience** — professional history, education, and certifications
- **Hobbies** — a peek at life outside of code (polymer clay, crocheting, cats)

## Tech stack

- React 18 + TypeScript
- Vite
- Tailwind CSS
- React Router
- Deployed via GitHub Pages

## Running locally

```bash
npm install
npm run dev
```

## Deploying

```bash
npm run build
npm run deploy
```

Deploys the `dist` folder to the `gh-pages` branch, served at [girlwhodrawsandstuff.github.io](https://girlwhodrawsandstuff.github.io).
